# Third-Party Files

This plugin includes Ulanzi SDK support files inside the installable plugin bundle.

## Ulanzi SDK

These folders are copied from the Ulanzi Deck Plugin SDK structure and should keep their SDK-compatible paths:

- `com.ulanzi.utilitysuite.ulanziPlugin/libs/`
- `com.ulanzi.utilitysuite.ulanziPlugin/plugin/plugin-common-node/`

They provide the property inspector UI helpers, WebSocket helpers, and Node service API used by Ulanzi Deck / Ulanzi Studio.

Project-specific code lives in:

- `com.ulanzi.utilitysuite.ulanziPlugin/plugin/app.js`
- `com.ulanzi.utilitysuite.ulanziPlugin/plugin/src/`
- `com.ulanzi.utilitysuite.ulanziPlugin/property-inspector/monitor-toggle.html`
- `com.ulanzi.utilitysuite.ulanziPlugin/scripts/WindowsDisplayControl.ps1`
- `com.ulanzi.utilitysuite.ulanziPlugin/scripts/WindowsDisplayWatcher.ps1`
